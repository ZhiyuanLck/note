# const限定符

## 存在的意义

- 定义常量

```cpp
const int a = 100;
```

类型为整数或枚举，且以常量表达式初始化

- 限制对变量的修改
- **const常量类型检查**
  - const常量具有类型，`#define`宏没有数据类型，不能进行安全检查
- **节省空间，避免不必要的内存分配**

const常量                    | `#define`
---------                    | ---------
给出了对应的内存地址         | 出的是立即数
在程序运行过程中只有一份拷贝 | 在内存中有若干个拷贝


## 重点

- 非const变量默认为extern，要使const变量能够在其他文件中访问，必须在文件中显式地指定它为extern
- 常量必须进行初始化
- 允许把非const对象的地址赋给指向const对象的指针（不能通过该指针修改）
- 不能使用void\*指针保存const对象的地址，必须使用const void\*类型的指针保存const对象的地址
- const指针必须进行初始化
