\section{系统配置}

\subsection{代理设置}
\subsubsection{socks转http}
安装 |proxychains4|
\begin{mysh}[|/etc/privoxy/config|]
  socks5 127.0.0.1 1080
\end{mysh}

\subsection{登录界面位置设置}
ubuntu设置：
\begin{mysh}
sudo cp ~/.config/monitors.xml ~gdm/.config/monitors.xml
sudo chown gdm:gdm ~gdm/.config/monitors.xml
\end{mysh}

\subsection{多显示器设置}
安装 |arandr| 将布局保存，每次启动载入布局
